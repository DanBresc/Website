Logic Gate Module

This is a Logic Gate Module that I created for download. It includes all of the basic logic gates as well as class that handles converty strings to binary and vice versa. The logic gates take two binary strings as inputs (with the exception of the not gate) and returns another binary string. I hope this can be a use to some programmers.

*If you encounter any confusion using the script. I wrote help text for every method. If confused just put the following into shell. >>>help(logicGates)

Here are some example uses:
>>> from logicGates import LogicGates
>>> LG = LogicGates()
>>> x = "10100"
>>> y = "10010"
>>> LG.AND(x,y)
'10000'
